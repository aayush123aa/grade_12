Xampp

Setting Up Xampp

  • Go to https://www.apachefriends.org/
  • Install the right file for your OS
  • Open the file and run it
  • Click next> if you don't want to make any changes to the download
  • Agree Terms and condition

Running XAMPP

  • Go to start menu and search "Xampp control panel"
  • Open Xampp control panel
  • Click on start SQL
  • Click on start apache
  • Open browser and search "https://localhost/phpmyadmin"
Note: Port is not required if you are using default ports

Managing Xampp

  • After starting sql and apache Open browser and search "localhost:(port)/phpmyadmin"
  • At server, go to sql and type "CREATE DATABASE DATABASE_NAME;"
  • Now open the database and go to sql again and type "CREATE TABLE TABLE_NAME(COLUMN1,COLUMN2,COLUMN3.....);"
  • You can add data by typing "INSERT INTO TABLE_NAME VALUES(VALUE1, VALUE2, VALUE3,.....),(VALUE1,VALUE2......);"
