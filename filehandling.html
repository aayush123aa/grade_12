File Handling

File handling in C basically refers to messing around with files all while only using codes. What this means, basically, is that the coder is basically bound to only the code editor and has no option but to write the long codes and execute them to write, read and edit files

Here are some questions to test your ability to write codes for file handling.

Question 1: Write a C program to write a character and display it using fgetc () and fputc () functions.

Question 2: Write a C program to write a number and display it using getw() and putw() functions.

Question 3: Write a C program to input a string and display it using fputs() and fgets() functions.
